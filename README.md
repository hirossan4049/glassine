# Glassine
[![SUSHI-WARE LICENSE](https://img.shields.io/badge/license-SUSHI--WARE%F0%9F%8D%A3-blue.svg)](https://github.com/MakeNowJust/sushi-ware)

![glassine-demo](./docs/assets/glassine-demo.gif)

ログイン不要の日程調整ツール

## 特徴

- ログイン不要で即座に利用可能
- URL共有のみのシンプルな運用
- ドラッグ&ペイントで直感的に日時選択
- ○△×で可否を入力、最適な時間帯を自動推薦

## クイックスタート

```bash
npm install
npm run dev
```
